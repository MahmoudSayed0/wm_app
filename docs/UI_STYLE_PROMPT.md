# Washman UI Style Prompt Guide

Use these prompts with AI design tools (Figma AI, Midjourney, DALL-E, etc.) to generate consistent UI designs.

---

## Master Prompt (English)

```
Design a premium mobile app UI screen with modern 3D soft morphism style.

Style characteristics:
- Soft, elevated 3D elements with subtle depth
- Gradient buttons (teal #1F8783 to dark teal #12504E)
- Light cream/off-white background (#FCFCFC)
- Smooth rounded corners (pill-shaped buttons, 12-16px cards)
- Soft drop shadows with slight blur (not harsh)
- Clean minimalist layout with generous white space
- Glassmorphism touches on cards (subtle transparency)
- Floating UI elements with layered depth effect
- Premium luxury feel, similar to high-end fintech apps
- iOS-style design language with soft UI elements
- Neumorphic input fields with inset subtle shadows
- Gradient text for accent elements
- Smooth color transitions, no harsh contrasts
- Touch-friendly large tap targets (44px minimum)
- Progress indicators with pill-shaped dots
- Dashed borders for special input fields (license plates)
- Category badges with gradient backgrounds

Color palette:
- Primary: Teal gradient (#1F8783 → #12504E)
- Background: Off-white (#FCFCFC)
- Text: Dark charcoal (#090909, #1A1A1A)
- Muted text: Gray (#6B7280, #9CA3AF)
- Input backgrounds: Light gray (#F9FAFB, #F2F2F2)
- Success: Green (#22C55E)
- Error: Red (#DC2626)

Typography:
- Clean sans-serif font (SF Pro, Inter, or similar)
- Bold headings (24-26px)
- Medium weight for buttons and labels
- Light weight for captions

The overall aesthetic should feel:
- Premium and trustworthy
- Modern and clean
- Soft and approachable
- Professional but friendly
- Similar to apps like Revolut, Wise, or premium car service apps
```

## Component-Specific Prompts

### Buttons
```
Pill-shaped gradient button, teal to dark teal horizontal gradient, soft drop shadow, white text, 16px vertical padding, 70px border radius, elevated 3D effect, premium mobile UI
```

### Input Fields
```
Neumorphic text input, light gray background (#F9FAFB), 12px rounded corners, 2px border, soft inset shadow effect, placeholder text gray, clean minimal style
```

### Cards
```
Floating card with soft shadow, white background, 16px rounded corners, subtle glassmorphism effect, layered depth, premium mobile UI, clean minimal content
```

### OTP Input
```
6 digit OTP input boxes, each box 45x45px, 8px rounded corners, light gray background, gradient text for entered digits, soft minimal design
```

### Bottom Sheet
```
iOS-style bottom sheet, 24px top rounded corners, white background, drag handle indicator, soft overlay backdrop, smooth slide animation, premium mobile UI
```

### Color Picker
```
Circular color swatches grid, 44px circles, thin border, checkmark icon on selected, soft shadow on selection, centered layout, car color palette
```
